About

Halo, perkenalkan saya Noufal Darias Akbar biasa dipanggil Noufal, saya merupakan salah satu mahasiswa Fakultas Teknik dan Ilmu Komputer di Universitas Indraprasta PGRI.

Sebagai mahasiswa Informatika dan ingin menjadi Web Developer, saat ini saya mempelajari berbagai macam bahasa pemrograman, salah satunya yaitu bahasa pemrograman web.

Bahasa pemrograman web yang sedang saya pelajari saat ini HTML, CSS, dan JavaScript. Selain bahasa pemrograman web, saya juga mempelajari berbagai macam Framework seperti Bootstrap, React JS, dan Laravel.

Activity

Membuat program atau coding menjadi salah satu aktivitas saya sehari-hari, dan juga untuk meningkatkan skill saya dalam membuat sebuah program.

Saya seorang mahasiswa aktif jurusan Teknik Informatika di Fakultas Teknik dan Ilmu Komputer Universitas Indraprasta PGRI.

Selain beraktivitas dirumah, terkadang saya juga beraktivitas diluar rumah, seperti riding keliling kota menggunakan sepeda motor yang sekaligus menjadi salah satu hobi saya.
